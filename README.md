# Graph Algorithms Research

## Description

This project demonstrates the implementation of basic graph algorithms using an adjacency matrix.

Implemented algorithms:

- Depth First Search (DFS)
- Breadth First Search (BFS)
- Dijkstra Shortest Path Algorithm

## Technologies

- C++
- Visual Studio 2022

## Graph Representation

The graph is stored as an adjacency matrix.

## Features

- Display adjacency matrix
- DFS traversal
- BFS traversal
- Shortest path calculation using Dijkstra algorithm
